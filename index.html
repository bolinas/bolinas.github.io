Rule Extraction and Parsing Tools for Synchronous Hyperedge Replacement Grammars

Welcome to Bolinas.

You can find our repository here: Bolinas

To clone the code (read-only), use the following command:

git clone git://github.com/karlmoritz/bolinas.git

If you are interested in contributing to the code, please fork a copy of the repository and then check-in any updates via a pull request.

About Bolinas

Bolinas is a set of tools for manipulating and using hyperedge replacement grammars. For more details on usage, see the documentation in the code - a longer README file and documentation are currently being prepared.
